# 前言

挑选自认为对游玩有帮助的工具站点贴出地址。打破信息隔膜！

## 信息新闻站
1. 【兑换码|新闻汇集】The BD2 Pulse ：[https://thebd2pulse.com/](https://thebd2pulse.com/)
    > 输入用户名一键兑换兑换码，缓存兑换状态。官方新闻、巴哈、NGA等轻量聚合。
2. 【新闻】官方繁体站：[https://www.browndust2.com/zh-tw/news](https://www.browndust2.com/zh-tw/news)
    > 官方公告站点，权威、访问速度慢。
3. 【付费性价比】由巴哈姆特[@蓝白社野怪 (fromzero1501)](https://home.gamer.com.tw/fromzero1501)持续追踪，货币单位**新台币**，大陆可能需解决连通性问题，[巴哈姆特-付费性价比目录贴](https://m.gamer.com.tw/forum/C.php?bsn=76207&page=&snA=3253&last=) 。
4. 更多待续
